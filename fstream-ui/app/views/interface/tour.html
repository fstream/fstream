How to use it

Tour is very easy to use. All you have to do is to define main tour container (in this example we define it on body element): tour backdrop="true". We also add a backdrop = true to add a gray background. Next you need to define a tour step with order, title, content and placement like: tour-step order="3" title="Tour title" content="Tour description" placement="right"

Example: 

<div class="main" tour backdrop="true">

      <div class="panel" tour-step order="1" title="Title" content="Tour Step Description" placement="top">
          This will be a step 1
      </div>
      <div class="panel" tour-step order="2" title="Title" content="Tour Step Description" placement="left">
          This will be a step 2
      </div>
      <div class="panel" tour-step order="3" title="Title" content="Tour Step Description" placement="right">
          This will be a step 3
      </div>
      <div class="panel" tour-step order="4" title="Title" content="Tour Step Description" placement="bottom">
          This will be a step 4
      </div>

</div>
Configuration

The TourConfigProvider allows you to set a couple options:

  • prefixOptions {boolean, default: false} if set to true will require directive options to be prefixed to avoid conflicts
  • prefix {string, default: 'bsTour'} the prefix to use if prefixOptions is set to true

Use TourConfigProvider.set(<option>, <value>) in your app's config block to change the settings

You can use either tour and tourStep or bsTour and bsTourStep as directive names without changing config.

See full documentation of official tour plugin: http://bootstraptour.com/ and on angular wrapper: https://github.com/benmarch/angular-bootstrap-tour.
