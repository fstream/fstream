Animation panel

Homer has special small directive animate-panel that allowes you to implement staggering animations to any layout elements. It is designed to work on any html elements in order to achive a effect of building layout on state change. It works also with ng-repeat but we recomand to use angular-animate to achive staggering effect with ng-repeat. This deirective is mainly for main layout elements.

It is easy to use. All you have to do is to add animate-panel to wrapper of elements that should be staggering. For example: 

<div animate-panel effect="zomeIn" child="element">

    <div class="element"></div>
    <div class="element"></div>
    <div class="element"></div>

</div>

In default directive get all .row > div elements and applay zoomIn effect. You can use any effect avalible with Animatin.css library. You can find many example of use in Homer theme but feel free to play with examples below.
